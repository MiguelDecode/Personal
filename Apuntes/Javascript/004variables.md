# Variables

- Son de tipado dinámico.
- Se dan dos fases en la creación de una variable y una a mayores en algunos casos. Por defecto las variables declaradas se les asigna el valor _undefined_.

```js
var nombre  //Declaración
nombre = 'Miguel' // Inicialización
nombre = 'Arya' // Reasignación
```
  
## var

- Es de ámbito global.
- Se ve afectada por el hoisting.

## let

- Es de ámbito de bloque.

## const

- Es de ámbito de bloque.
